TypeScript type definitions for [Better Genshin Impact](https://github.com/babalae/better-genshin-impact) JavaScript scripts.

## Installation

```shell
npm install --save-dev @bettergi/types
```

## Usage

```json
{
  "compilerOptions": {
    "types": ["@bettergi/types"]
  }
}
```
